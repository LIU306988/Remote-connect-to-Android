# ADB 设备管理器 2.1 便携版 — 使用说明

---

## 📖 简介

**ADB 设备管理器**是一款专为 Android 设备管理设计的图形化工具，集成了 **ADB** 和 **Scrcpy**，无需安装任何环境，开箱即用。支持 USB 和无线连接，提供设备管理、应用安装、屏幕投射等一站式功能。仅代码运行需安装Scrcpy4.0配置环境变量。

---

## ✨ 功能特点

- ✅ **纯绿色便携** — 单文件运行，无需安装，无需配置环境变量
- ✅ **多连接方式** — 支持 USB 有线连接和 WiFi 无线连接
- ✅ **设备管理** — 保存常用设备，一键连接，支持批量操作
- ✅ **屏幕投射** — 集成 Scrcpy，高清低延迟投屏，支持音频传输
- ✅ **应用管理** — 一键安装 / 卸载 APK，批量操作多设备
- ✅ **截图功能** — 一键截取设备屏幕，自动保存到本地
- ✅ **参数自定义** — 可自定义 Scrcpy 分辨率、码率、帧率等参数
- ✅ **操作日志** — 完整记录所有操作，方便问题排查

## 🚀 快速开始

### 第一步：开启 USB 调试

1. 打开手机 **「设置」→「关于手机」**
2. 连续快速点击 **「版本号」** 7 次，直到提示 *"您现在处于开发者模式"*
3. 返回设置主界面，找到 **「开发者选项」**（通常在「系统」或「更多设置」中）
4. 开启 **「USB 调试」** 开关
5. ⚠️ 重要：开启 **「USB 调试 (安全设置)」**（部分手机需要，否则无法用鼠标键盘控制）
6. 首次连接时，手机会弹出授权对话框 — **勾选「始终允许从此计算机进行调试」** 并点击确定

### 第二步：运行程序

1. 直接双击运行 `ADB设备管理器.exe`
2. ⚠️ 首次运行可能会被 **Windows Defender** 拦截，请点击 **「更多信息」→「仍要运行」**
3. 程序会自动在同目录生成两个配置文件：
   - `adb_devices.json` — 保存你的设备列表
   - `scrcpy_settings.json` — 保存你的 Scrcpy 设置

### 第三步：连接设备

#### 方式一：USB 有线连接

1. 用 USB 数据线将手机连接到电脑
2. 切换到 **「已连接设备」** 标签页
3. 点击 **「刷新连接列表」**，你的设备会显示在列表中

#### 方式二：WiFi 无线连接

1. 先用 USB 连接手机
2. 点击 **「获取有线设备 IP」** 按钮，程序会自动获取手机的 IP 地址
3. 点击 **「连接设备」** 按钮
4. 连接成功后，可以拔掉 USB 数据线

### 设备连接标签页

| 功能 | 说明 |
|------|------|
| 通过 IP 连接 | 输入设备的 IP 地址和端口（默认 5555）进行无线连接 |
| 获取有线设备 IP | 自动获取已通过 USB 连接的设备的 IP 地址 |
| 已保存的设备 | 显示所有已保存的设备，支持批量连接 |
| 设备管理 | 保存、删除、编辑、导入、导出设备列表 |
| 快速操作 | 一键连接选中设备，一键启动 Scrcpy 投屏 |

| 功能 | 说明 |
|------|------|
| 设备列表 | 显示所有当前已连接的设备及其状态 |
| 连接管理 | 刷新列表、断开选中连接 |
| 投屏控制 | 一键启动 Scrcpy 投屏 |
| **应用管理** | |
| └ 安装 APK | 选择本地 APK 文件安装到选中设备 |

| 类别 | 选项 |
|------|------|

| 音频设置 | 仅电脑、仅设备、两端同时、禁用 |

- 完整记录所有操作和错误信息
- 支持清空日志和导出日志到文件
- 方便问题排查和调试

## ❓ 常见问题解答

### Q1: 程序双击没反应怎么办？

**A:** 尝试以下方法：
- 右键点击程序，选择 **「以管理员身份运行」**
- 暂时关闭 Windows Defender 或其他杀毒软件
- 检查是否有其他 ADB 程序正在运行（如 Android Studio、手机助手等）

### Q2: 无法连接设备怎么办？

**A:**
- 确认手机已开启 **USB 调试** 和 **USB 调试 (安全设置)**
- 确认电脑和手机在同一局域网（无线连接时）
- 尝试更换 USB 数据线（建议使用原装数据线）
- 尝试更换电脑的 USB 端口
- 重启手机和电脑的 ADB 服务

### Q3: Scrcpy 投屏黑屏怎么办？

**A:**
- 尝试更换视频编码器（在 Scrcpy 设置中切换为 h264）
- 降低最大分辨率和比特率
- 确认手机已开启 **「USB 调试 (安全设置)」**
- 点击 **「测试默认参数」** 按钮查看详细错误信息

### Q4: 无法用鼠标键盘控制手机怎么办？

**A:**
- 确认手机已开启 **「USB 调试 (安全设置)」**
- 重启手机后重新连接
- 部分手机需要额外开启 **「允许通过 USB 调试模拟输入」** 选项

### Q5: 投屏没有声音怎么办？

**A:**
- 确认 Android 版本为 **11 及以上**（Scrcpy 音频功能要求）
- 在 Scrcpy 设置中检查音频模式是否设置为 **「仅电脑播放」**
- 检查电脑的音量是否开启

### Q6: 程序退出时提示 "Failed to remove temporary directory" 怎么办？

**A:**
- 这是正常现象，点击 **「确定」** 即可
- 可以手动删除 `C:\Users\你的用户名\AppData\Local\Temp\` 目录下的 `_MEIxxxxxx` 文件夹
- 建议使用目录模式打包，完全避免此问题

## ⚠️ 注意事项

- 🔒 **安全提示**：USB 调试模式有一定安全风险，不使用时请关闭
- 🛡️ **杀毒软件**：本程序可能会被杀毒软件误报病毒，这是 PyInstaller 打包程序的常见现象，请添加信任
- 🔗 **多设备连接**：同时连接多个设备时，操作前请确认选中了正确的设备
- 💾 **数据备份**：执行卸载应用等操作前，请务必备份重要数据
- 🔥 **防火墙设置**：无线连接时，请确保 Windows 防火墙允许本程序访问网络
